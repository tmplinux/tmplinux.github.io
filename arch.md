# tmparch
Quick and fast minimal distro. Please note that this does NOT support the AUR, as there is not another user and suid is disabled.

## Speed
### First time (bootstrap/download)
Downloading Packages - 31s
Setting stuff up - 10s
Squashing - 4s
Starting up - 3s
Login - 2s
Overall: 50s
### After first-time
Unsquashing - 0.73s
Booting - 2.53s
Logging in - 2.44s
Overall: 6s
